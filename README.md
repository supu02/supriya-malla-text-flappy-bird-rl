# Text Flappy Bird RL Assignment

This repository contains my submission for the Text Flappy Bird reinforcement learning assignment.

## Contents

- `Supriya_Malla_TFB_Assignment.ipynb`: main notebook for the submission
- `Supriya_Malla_TextFlappyBird_Report.pdf`: final report PDF
- `Supriya_Malla_TextFlappyBird_Report.tex`: report source
- `rl_assignment/`: supporting code and saved experiment results
- `vendor/text-flappy-bird-gym-master/`: local copy of the course Text Flappy Bird environment

## Running Locally

From the repository root:

```bash
pip install -r requirements.txt
jupyter notebook
```

The local environment package is installed from `vendor/text-flappy-bird-gym-master`.
